# Example Networks

The repository includes complete example network configs under `examples/schemas`.

## Yellow Dot

Path:

- `examples/schemas/yellow_dot/network.json`

Use case:

- education network

Domains:

- `student`
- `tutor`
- `coaching_center`

Actions:

- `connect`

## Blue Dot

Path:

- `examples/schemas/blue_dot/network.json`

Use case:

- jobs and hiring network

Domains:

- `seeker`
- `provider`

Actions:

- `apply`
